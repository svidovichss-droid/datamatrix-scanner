# Сканер качества печати DataMatrix

Программное обеспечение для контроля качества печати DataMatrix кодов на производственных линиях.

![Версия](https://img.shields.io/badge/version-1.0.0-blue)
![Python](https://img.shields.io/badge/python-3.10+-green)
![ГОСТ](https://img.shields.io/badge/standard-ГОСТ%20Р%2057302--2016-orange)

## Описание

Сканер предназначен для:
- Захвата изображений DataMatrix кодов с промышленных камер
- Автоматического распознавания DataMatrix кодов
- Оценки качества печати по методике ГОСТ Р 57302-2016
- Ведения истории сканирований
- Формирования отчётов и статистики

## Авторы

**А. Свидович / А. Петляков** для **PROGRESS**

## Требования

- Python 3.10 или выше
- USB или IP камера (поддержка OpenCV)
- Windows 10/11 x64

## Установка

### Вариант 1: Готовый exe файл (рекомендуется)

1. Скачайте последнюю версию с [Releases](https://github.com/YOUR_REPO/datamatrix-scanner/releases)
2. Запустите `DataMatrixScanner.exe`

### Вариант 2: Сборка из исходников

```bash
# Клонирование репозитория
git clone https://github.com/YOUR_REPO/datamatrix-scanner.git
cd datamatrix-scanner

# Установка зависимостей
pip install -r requirements.txt

# Запуск
python main.py
```

## Сборка exe файла

### Локальная сборка (Windows)

```bash
# Установите PyInstaller
pip install pyinstaller

# Выполните сборку
pyinstaller build.spec
```

### GitHub Actions (автоматическая сборка)

При пуше в репозиторий автоматически:
1. Собирается exe файл
2. Создаётся архив Release
3. Генерируются хеш-суммы

Подробности см. в `.github/workflows/build.yml`

## Использование

### Подключение камеры

1. Подключите промышленную камеру к USB порту
2. Запустите программу
3. Нажмите "Подключить камеру"

### Ручной захват

1. Нажмите "Захватить кадр" для получения текущего изображения
2. Программа автоматически найдёт DataMatrix код
3. Результаты анализа отобразятся на экране

### Автоматическое сканирование

1. Установите галку "Автозахват при обнаружении"
2. Нажмите "Начать сканирование"
3. Программа будет автоматически:
   - Захватывать кадры с камеры
   - Обнаруживать DataMatrix коды
   - Анализировать качество
   - Сохранять результаты в историю

## Метрики качества (ГОСТ Р 57302-2016)

| Метрика | Описание | Требование |
|---------|----------|------------|
| **Контраст** | Разница яркости модулей | ≥ 0.7 |
| **Rmax** | Макс. неравномерность | ≤ 50% |
| **ANE** | Ошибка неравномерности | ≤ 30 |
| **Целостность** | Сохранность структуры | ≥ 80% |
| **SNR** | Отношение сигнал/шум | ≥ 30 |

## Шкала оценок

| Оценка | Баллы | Описание |
|--------|-------|----------|
| A | 90-100 | Отлично |
| B | 70-89 | Хорошо |
| C | 50-69 | Удовлетворительно |
| D | 30-49 | Плохо |
| F | 0-29 | Непригоден |

## История и отчёты

Все сканирования сохраняются в локальную базу данных SQLite:
- Дата и время
- Расшифрованные данные
- Оценка качества
- Значения метрик
- Рекомендации

Экспорт в CSV:
1. Выберите период фильтрации
2. Нажмите "Экспорт"
3. Сохраните файл
