# Сканер качества печати DataMatrix - Modern Edition

Программное обеспечение для контроля качества печати DataMatrix кодов на производственных линиях.

![Версия](https://img.shields.io/badge/version-2.0.0-blue)
![Python](https://img.shields.io/badge/python-3.10+-green)
![ГОСТ](https://img.shields.io/badge/standard-ГОСТ%20Р%2057302--2016-orange)

## Описание

Сканер предназначен для:
- **Автоматического поиска** DataMatrix кодов в видеопотоке
- **Захвата лучших кадров** с интеллектуальным определением момента захвата
- **Распознавания и декодирования** DataMatrix кодов любой сложности
- Оценки качества печати по методике ГОСТ Р 57302-2016
- Ведения истории сканирований
- Формирования отчётов и статистики

## Ключевые особенности версии 2.0

- 🚀 **Автоматический автосканер** - непрерывный поиск и распознавание без участия оператора
- 🎯 **Множественные методы детекции** - L-образный паттерн, контуры, текстурный анализ
- 📸 **Умный захват кадров** - автоматический выбор лучшего кадра по стабильности и качеству
- ⚡ **Высокая производительность** - оптимизированные алгоритмы обработки в реальном времени
- 🔧 **Гибкая настройка** - регулируемые параметры детекции и захвата

## Использование

### Быстрый старт

```bash
# Запуск приложения
python main.py
```

### Подключение камеры

1. Подключите промышленную камеру к USB порту
2. Запустите программу
3. Нажмите **"Подключить камеру"** или выберите камеру из списка доступных

### Режимы работы

#### 1. Ручной захват

1. Нажмите **"Захватить кадр"** для получения текущего изображения
2. Программа автоматически найдёт DataMatrix код
3. Результаты анализа отобразятся на экране

#### 2. Автоматическое сканирование (рекомендуется)

1. Включите опцию **"Автосканер DataMatrix"**
2. Нажмите **"Начать сканирование"**
3. Программа будет автоматически:
   - Непрерывно анализировать видеопоток
   - Искать DataMatrix коды тремя методами детекции
   - Захватывать лучший кадр при стабильном обнаружении
   - Декодировать и сохранять результаты в историю

#### 3. Автозахват при обнаружении

1. Установите галку **"Автозахват при обнаружении"**
2. При обнаружении кода программа автоматически сохранит кадр
3. Изображение и данные добавятся в историю

### Настройка параметров детекции

В режиме автосканера доступны настройки:

- **Минимальный размер кода** - фильтр мелких ложных срабатываний
- **Максимальный размер кода** - ограничение для больших объектов
- **Порог контраста** - чувствительность к контрасту изображения
- **Количество стабильных кадров** - надёжность автозахвата

## Метрики качества (ГОСТ Р 57302-2016)

| Метрика | Описание | Требование |
|---------|----------|------------|
| **Контраст** | Разница яркости модулей | ≥ 0.7 |
| **Rmax** | Макс. неравномерность | ≤ 50% |
| **ANE** | Ошибка неравномерности | ≤ 30 |
| **Целостность** | Сохранность структуры | ≥ 80% |
| **SNR** | Отношение сигнал/шум | ≥ 30 |

## Архитектура автосканера

### AutoDataMatrixScanner

Основной класс автоматического сканера:

- **process_frame()** - обработка кадра, возврат результата
- **_detect_by_finder_pattern()** - поиск по L-образному маркеру
- **_detect_by_contours()** - поиск по квадратным контурам  
- **_detect_by_texture()** - текстурный анализ для сложных случаев
- **_decode_roi()** - декодирование с множественными попытками

### ContinuousScanner

Непрерывное сканирование в фоновом потоке:

- **start()** / **stop()** - управление потоком
- **process_frame_async()** - асинхронная обработка кадров
- **set_auto_capture()** - настройка автоматического захвата

### DataMatrixResult

Структура результата:

- **status** - статус (NOT_FOUND, FOUND, DECODED, ERROR)
- **data** - распознанные данные
- **bbox** - координаты рамки (x, y, w, h)
- **confidence** - уверенность детекции (0.0-1.0)
- **processing_time_ms** - время обработки
